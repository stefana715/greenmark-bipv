# PV × Art Paper Project

## Working topic
**Photovoltaics, Aesthetics, and Public/Architectural Art**

This subproject is created to evaluate whether the current PV × art materials can be developed into:
1. an SCI paper, or
2. a Chinese core-journal paper.

---

## Current assessment
Based on the uploaded archive inventory, the material package appears to include:
- one PowerPoint: `PV with Art.pptx`
- one PDF reference file
- multiple visual case examples related to solar art / coloured PV / BIPV / public installations

Examples visible from filenames include:
- Sarah Hall projects
- Loop.pH / Sonumbra
- coloured solar panels
- solar gate / solar sail / Zadar solar monument related images

At this stage, the package looks more like a **case-reference set** than a finished research dataset.

---

## Most viable paper routes

### Route A — SCI review / framework paper (recommended)
**Possible title direction:**
"From Building-Integrated Photovoltaics to Public Solar Art: A review and assessment framework for energy-aesthetic integration"

**Why this route is viable:**
- current material is rich in case references
- the topic is interdisciplinary and timely
- coloured/aesthetic BIPV literature is growing quickly
- a review + framework paper does not require full lab experimentation at the first stage

**Core contribution should be:**
- not just collecting examples
- but building a reproducible framework that compares cases across:
  - energy function
  - integration type
  - aesthetic strategy
  - public interaction
  - urban/place value
  - cost / feasibility / maintainability

### Route B — SCI empirical case paper (possible, but harder)
This route would require at least one of the following:
- measured performance data
- simulation / LCA / techno-economic comparison
- user perception survey
- expert evaluation / Delphi / AHP / SEM or similar method

Without those, it will be difficult to convince a strong SCI journal.

### Route C — Chinese core-journal paper (safe fallback)
**Possible direction:**
A theoretical + case-based paper on PV-art integration, aesthetic value, and design strategy.

This route is realistic even if the current package remains mainly visual and conceptual.

---

## Initial research questions
1. How can photovoltaic systems move from pure energy devices to architectural and artistic media?
2. What design strategies are used to reconcile energy output and aesthetic quality?
3. What dimensions are missing in current BIPV evaluation beyond efficiency and cost?
4. Can a structured assessment framework be built for PV-art / aesthetic BIPV cases?

## What is needed next
To upgrade this project into a publishable paper, the next step is to convert the current inspiration material into structured evidence:
- extract all identifiable cases from the PPT/PDF
- build a case matrix
- define evaluation dimensions
