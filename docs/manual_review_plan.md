## What to extract from each manual

For every PV/BIPV-related credit found, record:

1. **Credit ID and name** (e.g., "EAc5 Renewable Energy")
2. **Category it belongs to** (e.g., Energy & Atmosphere)
3. **Maximum points available** for this credit
4. **Total points in the system** (to calculate %)
5. **Requirement type**: Performance (% energy) / Prescriptive (kWp installed) / Process (feasibility study)
6. **Exact language about PV**: Does it say "renewable energy" / "solar" / "PV" / "BIPV" / "building-integrated"?
7. **BIPV application types mentioned**: Roof / Façade / Window / Shading / None specified
8. **Whether BIPV is distinguished from BAPV**: Yes / No
9. **Prerequisites/mandatory requirements** related to PV
10. **Innovation/bonus credits** available for PV/BIPV
11. **Post-occupancy verification** requirements for PV output
12. **Cross-references** to other categories (e.g., does PV also help with envelope, materials, daylighting credits?)
13. **Version history**: How have PV criteria changed from previous versions?
